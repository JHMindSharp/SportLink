L'équipe

Jonathan Huybrechts

Développeur Full Stack, Chef de projet, Designer

En tant que seul développeur, je suis responsable de tous les aspects du développement de l'application, y compris le backend, le frontend, le design et l'interaction utilisateur. Ce rôle exploite mon ensemble de compétences pour gérer et exécuter l'ensemble du cycle de vie du projet.

Technologies utilisées

  • Langages : Python, JavaScript
  • Frameworks : Flask pour le backend, React Native pour le frontend
  • Bases de données : MongoDB pour stocker les profils utilisateurs, les réservations et les statistiques de santé
  • APIs : Développement d'une API interne pour le suivi des statistiques de santé et la gestion des profils
  • Paiements : Intégration de PayPal et Apple Pay pour des transactions sécurisées
  • Communication en temps réel : WebSocket pour la messagerie instantanée entre utilisateurs
  • Autres outils : Git, GitHub, AWS
